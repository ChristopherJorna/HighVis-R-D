# P002 — Camera-locked video synthesis, first frame + source video

**Created:** 2026-04-09
**Target model:** Video gen (Luma / compare)
**Type:** Video generation — first frame + untextured source video → realistic video

## Prompt
```
Use First Frame as the exact first frame. Apply the identical camera movement from [source video] with zero deviation. Match camera position, camera rotation, camera speed, easing, and timing exactly to the source video. The scene must behave as if the same camera rig was reused on the same tripod.

All characters, subjects, and objects visible in First Frame must remain in the exact same pose, silhouette, proportions, orientation, and world-space position throughout the entire shot. No pose changes, no limb movement, no body rotation or translation, no scale changes, and no procedural motion.

Background and environment constraints: The background, environment, and all non-character elements must remain exactly the same as in First Frame. No new background elements may be introduced. No racetrack, road, scenery, buildings, props, set extensions, or environmental changes are allowed. Do not replace, enhance, reinterpret, or stylize the background. Background must remain identical in layout, scale, perspective, lighting direction, and spatial relationship to the subject.

The only change allowed is improved photorealism and surface detail. Preserve original composition, horizon line, perspective, and framing at all times.

Hard constraints: Camera transform must remain unchanged throughout the shot. Camera path is immutable. Any deviation from the original camera motion, character pose, background, or allowed facial motion is not allowed.

Hyper realistic, copy the exact style of the first frame.
```

## How it was built
- Built to enforce camera lock — video models tend to drift camera/pose without explicit constraints
- "Hard constraints" section repeated intentionally for emphasis
- References First Frame (P001 output) as style anchor
- "Immutable" / "not allowed" language used to fight model hallucination tendencies
